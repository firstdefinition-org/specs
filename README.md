# FirstDefinition Specifications

This repository is a read-only mirror of specifications published at:

https://firstdefinition.com/specs

The canonical versions of all specifications live on the FirstDefinition domain.
In the event of any discrepancy, the website versions are authoritative.

This repository exists to provide:
- long-term persistence
- version history
- diff visibility

Issues and pull requests may be disabled or unmonitored.
